# **vertex STARK**

A clear and educational Python-based implementation of **STARKs
(Scalable Transparent ARguments of Knowledge)**, along with foundational
tools and components used in modern zero-knowledge proof systems.

This repository walks through the core building blocks of STARKs, from
algebraic foundations to full interactive proofs, making it ideal for
learners, researchers, and practitioners exploring transparent proof
systems.

------------------------------------------------------------------------

## **üìò Outline**

### **1. Introduction**

An accessible introduction to the motivation behind STARKs,
transparency, and proof scalability.

### **2. Overview of STARKs**

A high-level breakdown of: - Arithmetization\
- Low-degree testing\
- Commitment schemes\
- Interactive oracle proofs

### **3. Basic Tools --- Algebra & Polynomials**

Essential mathematical foundations including: - Finite fields\
- Polynomial operations\
- Evaluation domains\
- Interpolation

### **4. FRI: Low-Degree Testing**

A walkthrough of the **Fast Reed--Solomon Interactive Oracle Proof
(FRI)** protocol: - Folding steps\
- Query phases\
- Soundness intuition

### **5. STARK Information-Theoretic Protocol**

Construction of a STARK proof system: - Execution trace\
- Constraint system\
- Commitment phases\
- Query phases\
- Verification logic

### **6. Performance Improvements: NTT & Preprocessing**

Speeding up the protocol using: - Number Theoretic Transform (NTT)\
- Domain preprocessing\
- Efficient polynomial operations
